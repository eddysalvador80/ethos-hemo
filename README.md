# ETHOS·HEMO Materna

Apoyo a la decisión clínica sobre **sangre y coagulación en la embarazada** (hemorragia, anemia, anticoagulación/neuraxial, eclampsia, trombocitopenia, hemorragia anteparto). Determinista, **offline**, con bibliografía verificada. App web instalable (PWA).

**App:** https://eddysalvador80.github.io/ethos-hemo/

No reemplaza el juicio clínico ni al equipo. Default = protocolo local SSASUR/HHHA; discrepancias anotadas.
